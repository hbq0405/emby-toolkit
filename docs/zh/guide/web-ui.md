# Web 控制台

Web UI 由 `emby-actor-ui` 构建并由后端提供静态资源，入口默认是：

- `http://<服务器IP>:5257`

## 主要功能区

- **系统状态**：任务运行状态、日志、运行信息。
- **配置管理**：保存 Emby/TMDb/豆瓣/AI/监控/任务链等配置。
- **任务触发**：执行一次性任务、任务链与停止任务。
- **订阅管理**：追剧、演员订阅、自建合集管理入口。
- **日志查看**：结合后端日志队列查看处理进度。

## 权限与认证

- Emby管理员账号登录可完整管理后台。
- Emby普通用户登录只可访问用户中心和影视探索。
