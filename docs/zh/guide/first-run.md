# 首次配置

首次启动后，需要在 Web 控制台完成基础配置，建议顺序如下。

## 1. Emby 连接

- Emby 服务器地址（内网/局域网地址）
- Emby API Key
- Emby 用户 ID（用于操作/同步）
- 可选：管理员用户名/密码（用于自动获取令牌）

## 2. TMDb 与豆瓣

- TMDb API Key（必填）
- TMDb 语言偏好（如 `zh`）
- 豆瓣 Cookie（可选，提升豆瓣数据可用性）

## 3. MoviePilot（可选）

用于订阅与缺失资源补全：

- URL / 用户名 / 密码
- 智能订阅开关与请求频率限制

## 4. 任务链与监控

- 开启任务链，并设置 Cron
- 监控目录与排除目录
- 监控扩展名与回溯天数

## 5. Webhook

在 Emby 中配置 Webhook：

- 地址：`http://<服务器IP>:5257/webhook/emby`
- 内容类型：`application/json`
- 建议事件：新媒体添加、媒体删除、媒体元数据更新、媒体图像更新
