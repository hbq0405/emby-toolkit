# 首次配置

首次启动后，需要在 Web 完成初始化基础配置，填入EmbyURL和API密钥保存后即可用Emby管理员账户登录。

## 1. 通用设置

- 基础设置：按需配置
- 实时监控：建议开启
- 数据源与API：必填

## 2. Emby & 虚拟库

- Emby 连接设置：按需填写
- 虚拟库：按需启用

## 3. 智能服务

- AI翻译：中文化必填
- MoviePilot订阅：有订阅需求必填

## 4. 高级

- 网络代理：必填，除非你有全局代理
- 日志配置：按需配置
- 数据管理：数据维护模块
- 通知设置：配置后可接收入库通知

## 5. 115 网盘

如果你要使用整理、STRM、播放直链或共享资源，需要配置 115 网盘。

建议先完成：

- 登录授权，用于文件管理和整理。
- Cookie 扫码，用于播放直链。
- 待整理目录、未识别目录、媒体库根目录。
- 本地 STRM 根目录和 ETK 服务器地址。

详细说明见 [115 网盘集成](/zh/guide/p115)。

## 6. 共享资源中心

共享资源中心不是必须开启的功能。建议在本地入库、115、TMDb 都正常后再启用。

开启后你可以：

- 搜索中心资源库。
- 秒传或转存别人共享的资源。
- 把自己的资源登记到中心。
- 使用贡献点和求共享。

详细说明见 [共享资源中心](/zh/guide/shared-resource)。
