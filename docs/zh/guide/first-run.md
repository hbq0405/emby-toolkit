# 首次配置

首次启动后，需要在 Web 完成初始化基础配置，填入EmbyURL和API密钥保存后即可用Emby管理员账户登录。

## 1. 通用设置

- 基础设置：按需配置
- 实时监控：建议开启
- 数据源与API：必填

::: warning Emby 媒体库硬性要求
这里的“实时监控”指 ETK 自己的监控，不是 Emby 媒体库里的实时监控。

使用 ETK 管理媒体库时，Emby 媒体库必须满足两个条件：

- 关闭 Emby 媒体库实时监控，尤其是 STRM 媒体库。
- 开启 Emby 的 NFO 读取，让 Emby 优先读取 ETK 生成的 NFO 元数据。

ETK 需要先完成整理、重命名、STRM、NFO、海报、媒体信息等处理，再统一通知 Emby 扫描。Emby 实时监控一旦开启，就可能绕过 ETK 抢先扫描中间态文件；没有开启 NFO 读取时，Emby 也无法正确吃到 ETK 写好的元数据。
:::

## 2. Emby & 虚拟库

- Emby 连接设置：按需填写
- 虚拟库：按需启用

## 3. 智能服务

- AI翻译：中文化必填
- MoviePilot订阅：有订阅需求必填

## 4. 高级

- 网络代理：必填，除非你有全局代理
- 日志配置：按需配置
- 数据管理：数据维护模块
- 通知设置：配置后可接收入库通知

## 5. 115 网盘

如果你要使用整理、STRM、播放直链或共享资源，需要配置 115 网盘。

建议先完成：

- 登录授权，用于文件管理和整理。
- Cookie 扫码，用于播放直链。
- 待整理目录、未识别目录、媒体库根目录。
- 本地 STRM 根目录和 ETK 服务器地址。

详细说明见 [115 网盘集成](/zh/guide/p115)。

## 6. 共享资源中心

共享资源中心不是必须开启的功能。建议在本地入库、115、TMDb 都正常后再启用。

开启后你可以：

- 搜索中心资源库。
- 秒传或转存别人共享的资源。
- 把自己的资源登记到中心。
- 使用贡献点和求共享。

详细说明见 [共享资源中心](/zh/guide/shared-resource)。
