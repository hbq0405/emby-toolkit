# 实时监控

实时监控新增媒体项。

## 关键配置
![alt text](/public/images/ssjk.png)
- **监控路径**：需要监控的路径列表
- **排除路径**：需要排除的路径列表，必须是监控路径的子路径，命中则跳过刮削仅刷新
- **定时扫描回溯天数**：扫描监控目录任务将只扫描此配置天数内有变化的文件
- **监控扩展名**：仅监控配置了扩展名的文件类型
- **图片语言偏好**：刮削时将优先下载配置的语言图片

## 流程说明

监控到有新文件，程序会立即进行并发刮削、中文化、统一映射美国分级，处理完成会写入覆盖缓存并通知Emby扫描入库

## 常见建议

- 监控目录保持与 Emby 媒体库路径一致。
- 排除“临时下载/缓存”目录，避免误触发。
