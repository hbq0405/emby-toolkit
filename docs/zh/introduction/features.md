# 功能特性

以下功能来自仓库当前实现（`tasks/`、`routes/`、`core_processor.py`、`monitor_service.py` 等模块）。

## 媒体元数据增强

- **演员/角色中文化**：演员名与角色名翻译，可结合 AI 翻译与豆瓣角色名优先策略。
- **多数据源补全**：TMDb 与豆瓣信息补全，支持本地元数据 JSON（神医模式）。
- **评分与标签同步**：可将评分/标签等信息回写到 Emby。

## 订阅与智能追剧

- **智能追剧**：自动追踪系列状态，补全缺失季/集并发起订阅。
- **演员订阅**：追踪演员作品，按配置自动订阅历史与未来资源。
- **统一订阅处理**：与 MoviePilot 联动完成订阅与请求动作。

## 合集与虚拟库

- **原生合集补全**：自动识别并维护 TMDb 合集。
- **自建合集**：支持榜单类与规则筛选类合集，虚拟呈现在 Emby 中。
- **反向代理展示**：通过代理层将虚拟库与原生库合并展示。

## 运行与调度

- **任务链**：高频/低频任务链，按 Cron 配置执行。
- **后台队列**：任务互斥、状态可视化，支持中止。
- **实时监控**：媒体目录新增/移动/删除事件驱动处理。

## 运维与体验

- **封面生成**：为媒体库与合集生成统一风格封面。
- **通知与消息**：支持 Telegram 通知新入库或追更事件。
- **用户管理**：同步 Emby 用户与权限缓存。
