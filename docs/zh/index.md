# Emby Toolkit 文档

Emby Toolkit 是一个面向 Emby 媒体库的增强管理工具，围绕元数据补全、演员/角色翻译、合集与订阅、智能追剧、封面生成、实时监控等场景提供自动化能力，并带有 Web 管理界面与任务调度系统。

## 项目定位

- 目标：提升 Emby 媒体库的完整度、统一性与自动化运营能力。
- 方式：通过任务链、实时监控、Web API 与外部数据源集成，持续补全并维护元数据与合集结构。
